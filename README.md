<h1>Recast 🪄</h1>

You can switch between three font families and adjust background colors to match your preference. If a site doesn’t offer dark mode or the typography feels off, you can change it instantly.

It’s about making the web feel more comfortable to read on your terms.

<h3>How to use</h3>

1. Open any webpage in Chrome
2. Click the Recast extension
3. Choose a font, choose a background
4. The page updates instantly
5. Save your combo
6. Reset to return to the original styles

<h3>Fonts</h3>

1. Sans-serif
2. Serif
3. Mono

<h3>Backgrounds</h3>

1. Light #FAF9F6
2. Dark #2C2C2C
4. Paper #F5F0E8

<h3>How it works</h3>

The extension applies a new style to the page based on what you select.

It adds a font and a background to the current webpage
It removes the previous style before applying a new one
Clicking options in the UI updates the page instantly

I also started building a preset feature:

- You can save font and background combinations
- Presets are stored and shown in the UI
- Clicking a preset applies those same styles again
